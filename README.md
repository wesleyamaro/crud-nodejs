# Test
CRUD using NodeJs, Express and MongoDB. <br />

## Clone repository
```git clone https://github.com/WesleyAmaro/test.git```

## Install dependencies of the project
```npm install```

## Install MongoDb
* https://www.mongodb.com/download-center?jmp=nav#community
* https://robomongo.org/

## Gulp
```gulp```

## Running server
```npm run dev```
